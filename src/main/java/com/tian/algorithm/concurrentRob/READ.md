

# 抢优惠券实现
题目背景：
1. 定义不同场景条件下（货品种类，交易金额，交易时间段）可用的营销优惠券，及优惠金额计算规则（固定金额，按比例）
2. 在多线程访问的场景下，内存中维护一组上述营销优惠券，及相应的可用数量
3. 支付收银台咨询可用优惠券，并按优惠金额降序排序。
产出：
1. 一个通过Java main函数可执行，输出结果的代码
